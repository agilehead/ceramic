Ceramic!

Based on JSON-Schema, ceramic can map simple JSON into domain entities used in the application. It can also validate entities against constraints specified in the schema.

Ceramic also comes with two optional batteries; Ceramic-db, an Object Document Mapper (ODM) and Ceramic-http-parser, a parser which converts HTTP form data to entities.
